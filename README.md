# DJDX

A score viewer for beatmania IIDX, SOUND VOLTEX, and Polaris Chord, built for iOS.

Features:
- Full English and Japanese support
- Beautifully crafted UI built for iOS devices
- Sort, filter, search, and go back in time
- Customizable visual analytics
- Automatic import using WKWebView, including dark mode support for the KONAMI login page
- Manual import using CSVs (for beatmania IIDX and SOUND VOLTEX only)

Wording Clarifications:
- In beatmania IIDX, **DIFFICULTY** refers to the top level category (HYPER, ANOTHER, etc), while **LEVEL** refers to the lower level difficulty number (1 to 12).
